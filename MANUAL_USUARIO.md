# Manual de Usuario - VIDA A TUS PIES

## 1. Inicio de Sesión
Al abrir la aplicación, verás la pantalla de inicio de sesión.
- Ingresa tu correo electrónico registrado.
- En modo DEMO, puedes usar cualquier contraseña.
- **Nota:** El sistema detecta automáticamente si eres Administrador o Podólogo.

## 2. Dashboard (Panel Principal)
Aquí verás un resumen rápido de la clínica:
- **Estadísticas:** Número de pacientes, citas del día, ingresos estimados.
- **Próximas Citas:** Lista de las citas programadas para el día de hoy.
- **Acciones Rápidas:** Botones para crear una nueva cita o registrar un paciente rápidamente.

## 3. Agenda
El módulo de agenda te permite visualizar y gestionar las citas.
- **Vistas:** Puedes ver la agenda de **Todas las Sedes**, o filtrar por **Sede Norte** / **Sede Sur**.
- **Crear Cita:** Haz clic en el botón "Nueva Cita". (En esta versión Demo, los datos se guardan temporalmente en tu navegador).
- **Detalles:** Pasa el mouse sobre una cita para ver los detalles rápidos (Paciente, Servicio, Doctor).

## 4. Pacientes
Gestión de fichas médicas.
- **Buscar:** Usa la barra de búsqueda para encontrar pacientes por nombre o cédula.
- **Historial:** Visualiza el historial clínico básico y datos de contacto.
- **Nuevo Paciente:** Registra nuevos pacientes en la base de datos.
- **Ficha Completa:** Haz clic en "Ver Ficha" para ver el historial detallado y adjuntos.

## 5. Servicios
Catálogo de precios y tratamientos.
- Visualiza la lista de servicios configurados.
- Cada tarjeta muestra la **Duración**, **Precio** y en qué **Sedes** está disponible.
